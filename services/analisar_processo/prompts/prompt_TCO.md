# PROMPT DE ANÁLISE — Termo Circunstanciado (TCO)

Rito sumaríssimo da Lei 9.099/95. Pena máxima ≤ 2 anos.
Fluxo: composição → transação → denúncia oral → sursis → AIJ → sentença.

## ⚠️ PRESCRIÇÃO — MAIOR RISCO
Contravenções prescrevem em 2 anos. Se TCO parado > 2 anos → PROVAVELMENTE PRESCRITO.
