# Modelos de Despacho

> Uso: campo modelo_despacho do CSV. Adaptar ao caso concreto.

## 1. Receber Denúncia e Citar (Art. 396 CPP)
```
Recebo a denúncia oferecida pelo Ministério Público em face de [NOME], como incurso(a) nas sanções do Art. [ARTIGO].
Cite-se o(a) réu(é) para apresentar resposta à acusação no prazo de 10 (dez) dias (Art. 396 c/c 396-A do CPP).
Caso não seja localizado(a), cite-se por edital com prazo de 15 (quinze) dias.
Intimem-se.
```

## 2. Rejeitar Denúncia (Art. 395 CPP)
```
Rejeito a denúncia oferecida em face de [NOME], com fundamento no Art. 395, [I/II/III], do CPP, [por ser inepta / faltar pressuposto processual / faltar justa causa].
[FUNDAMENTAÇÃO].
Arquivem-se.
```

## 3. Nomear Defensor Dativo (Art. 396-A §2o)
```
Decorrido o prazo sem resposta, nomeio defensor(a) dativo(a) para apresentar defesa no prazo de 10 (dez) dias (Art. 396-A, §2o, do CPP).
Intime-se a Defensoria Pública.
```

## 4. Absolvição Sumária (Art. 397)
```
ABSOLVO SUMARIAMENTE o(a) réu(é) [NOME], com fundamento no Art. 397, [I/II/III/IV], do CPP.
[FUNDAMENTAÇÃO].
Transitada em julgado, arquivem-se.
```

## 5. Designar AIJ (Art. 399)
```
Não sendo caso de absolvição sumária, designo AIJ para [DATA], às [HORA].
Intimem-se o MP, a defesa, o(a) réu(é), testemunhas e vítima.
```

## 6. Redesignar AIJ
```
Redesigno a AIJ para [DATA], às [HORA].
Intimem-se as partes e testemunhas.
```

## 7. Intimar para Alegações Finais (Art. 403)
```
Encerrada a instrução, intimem-se as partes para alegações finais por memoriais, prazo sucessivo de 5 dias, iniciando pelo MP (Art. 403, §3o, do CPP).
```

## 8. Abrir Vista ao MP (Art. 46)
```
Dê-se vista ao Ministério Público para manifestação no prazo de [15/30] dias.
```

## 9. Reintimar MP
```
Decorrido o prazo sem manifestação ministerial, reintime-se o MP no prazo de [15] dias.
```

## 10. Homologar Arquivamento
```
Homologo o arquivamento do inquérito policial, nos termos da promoção ministerial.
Registre-se. Dê-se baixa.
```

## 11. Intimar Delegado para Relatório
```
Intime-se a autoridade policial para relatório conclusivo no prazo de [10/30] dias.
```

## 12. Processar Apelação (Art. 600)
```
Recebo a apelação interposta por [PARTE].
Intime-se a parte contrária para contrarrazões no prazo de 8 dias (Art. 600 do CPP).
Após, remetam-se ao Eg. TJBA.
```

## 13. Revisar Prisão Preventiva (Art. 316 §unico)
```
Em cumprimento ao Art. 316, §unico, do CPP, reviso a prisão preventiva de [NOME].
[Mantenho: subsistem os fundamentos, notadamente [FUNDAMENTO].]
[OU: Revogo. Expeça-se alvará clausulado, mediante [CAUTELARES].]
```

## 14. Homologar ANPP (Art. 28-A)
```
Homologo o ANPP celebrado entre o MP e [NOME], nos termos do Art. 28-A do CPP.
Condições: [LISTAR]. Prazo: [PRAZO].
Suspendo o processo. Cumpridas as condições, será declarada extinta a punibilidade.
```

## 15. Reconhecer Prescrição
```
Reconheço a extinção da punibilidade de [NOME] pela prescrição, relativamente ao crime de [CRIME] (Art. [ARTIGO]), nos termos do Art. 109, [INCISO], c/c Art. 107, IV, CP.
[Último marco interruptivo em [DATA] + prazo de [X] anos = [DATA LIMITE], já transcorrido.]
Transitada em julgado, dê-se baixa e arquivem-se.
```

## 16. Audiência Preliminar (9.099/95)
```
Designo audiência preliminar (Art. 72 da Lei 9.099/95) para [DATA], às [HORA].
Intimem-se as partes e vítima. Informe-se sobre composição civil e transação penal.
```

## 17. Sursis Processual (Art. 89 Lei 9.099)
```
Homologo a suspensão condicional do processo (Art. 89 da Lei 9.099/95).
Período de prova: [2/3/4] anos.
Condições: reparação do dano; proibição de frequentar [LUGARES]; comparecimento [mensal] ao juízo; nao ausentar-se sem autorização.
Suspendo processo e prescrição.
```

## 18. Revogar Sursis Processual
```
Revogo a suspensão condicional (Art. 89, §[3o/4o/5o], Lei 9.099/95), [por novo crime / nao reparação / descumprimento].
Retome-se a ação penal. Designo AIJ para [DATA].
```

## 19. Extinção por Sursis Cumprido
```
Cumprido o período de prova sem revogação, declaro extinta a punibilidade de [NOME] (Art. 89, §5o, Lei 9.099/95).
Transitada em julgado, arquivem-se.
```

## 20. Suspensão Art. 366 CPP
```
Citado(a) por edital, [NOME] nao compareceu nem constituiu advogado.
Suspendo o processo e a prescrição (Art. 366 CPP).
[Determino produção antecipada de provas urgentes: ESPECIFICAR, se houver.]
```

## 21. Deferir Medida Protetiva (Lei 11.340)
```
Defiro as MPUs em favor de [VÍTIMA] (Art. 22 da Lei 11.340/06).
Determino ao requerido [AGRESSOR]: [afastamento do lar / proibição de aproximação a [X] metros / proibição de contato].
Fica advertido de que o descumprimento configura crime (Art. 24-A, pena 1-2a detenção) e pode ensejar preventiva.
Expeça-se mandado.
```

## 22. Converter IP em Ação Penal — Tráfico
```
Recebo a denúncia em face de [NOME], como incurso(a) no Art. [33/35] da Lei 11.343/06.
[Mantenho preventiva / Concedo liberdade provisória mediante cautelares Art. 319.]
Cite-se para resposta em 10 dias.
```

## 23. Pronunciar Réu (Art. 413)
```
Pronuncio [NOME] como incurso(a) no Art. [121, §2o, INCISOS] do CP, para julgamento pelo Tribunal do Júri.
[Materialidade + indícios de autoria. Qualificadoras.]
[Manter/revogar prisão.]
Intimem-se. Prazo de 5 dias para arrolar testemunhas de plenário (max 5).
```
