# Crimes Contra o Patrimônio

> Natureza: CRIMINAL | Base legal: Art. 155-183 CP
> Dependências: -> criminal/prescricao.md | -> criminal/dosimetria.md (para privilegiado)

## Furto (Art. 155 CP)

| Tipo | Pena |
|------|------|
| Simples | 1-4a reclusão + multa |
| Noturno | +1/3 |
| Qualificado (§4o) | 2-8a reclusão + multa |
| Qualificado — explosivo (§4o-A) | 4-10a reclusão + multa (hediondo) |
| Qualificado — fraude eletrônica (§4o-B) | 4-10a reclusão + multa |
| Veículo p/ outro estado (§6o) | 2-8a reclusão + multa |

Qualificadoras §4o: destruição de obstáculo, abuso de confiança, escalada, destreza, chave falsa.

### Privilegiado (Art. 155 §2o)
Reu primário + coisa de pequeno valor (até 1 SM): juiz pode substituir por multa, reduzir 1/3 a 2/3, ou aplicar só detenção.

### Insignificância
Requisitos STF/STJ: mínima ofensividade, nenhuma periculosidade social, reduzido grau de reprovabilidade, inexpressividade da lesão jurídica. Reincidente: jurisprudência diverge.

## Roubo (Art. 157 CP)

| Tipo | Pena |
|------|------|
| Simples | 4-10a reclusão + multa |
| Majorado §2o | +1/3 a 1/2 |
| Majorado §2o-A (arma de fogo) | +2/3 |
| Majorado §2o-A (arma restrita/explosivo) | Dobro |
| Lesão grave (§3o I) | 7-18a reclusão + multa |
| Latrocínio (§3o II) | 20-30a reclusão + multa (hediondo) |

Majorantes §2o: arma branca, concurso de agentes, veículo, restrição de liberdade da vítima.

Sumula 582 STJ: consuma-se com violência/grave ameaça, mesmo sem posse mansa e pacífica.
Sumula 610 STF: latrocínio consuma-se quando há morte, ainda que nao haja subtração.
Sumula 443 STJ: aumento pela quantidade de majorantes exige fundamentação concreta.

## Extorsão (Art. 158 CP)
4-10a reclusão + multa. Com restrição de liberdade: 6-12a. Lesão grave: 7-18a. Morte: 20-30a (hediondo).

## Estelionato (Art. 171 CP)
1-5a reclusão + multa.

### Ação penal (Art. 171 §5o)
Regra: condicionada à representação. Exceções (incondicionada): Administração Pública, criança/adolescente/idoso/incapaz, estelionato eletrônico.

### Estelionato eletrônico (Art. 171 §2o-A)
Fraude via dispositivo eletrônico/informático. 4-8a reclusão + multa. Majorado se servidor fora do país: +1/3 a 2/3.

## Receptação (Art. 180 CP)

| Tipo | Pena |
|------|------|
| Simples | 1-4a reclusão + multa |
| Qualificada — atividade comercial | 3-8a reclusão + multa |
| Culposa | 1m-1a detenção + multa |

## Apropriação Indébita (Art. 168 CP)
1-4a reclusão + multa. Majorado em 1/3: depósito necessário, tutor/curador, transportador, depositário judicial.

## Dano (Art. 163 CP)
Simples: 1-6m detenção + multa. Qualificado: 6m-3a detenção + multa.

## Invasão de Dispositivo Informático (Art. 154-A CP)
1-4a reclusão + multa. Qualificado (prejuízo econômico): 2-5a reclusão + multa.

## Para a Análise
- Latrocínio/extorsão com morte: CRITICA (hediondo)
- Roubo com arma de fogo: ALTA
- Furto qualificado com explosivo: ALTA (hediondo)
- Estelionato simples: verificar prescrição e representação
- Furto simples de pequeno valor: verificar insignificância
- Réu menor de 21 na data: prescrição pela metade (-> criminal/prescricao.md)
